# Functions, Modules, Objects

## Exercise 1:
Write a Python function that prompts the user to enter a number.
Given that number, the function should print the sentence “Hello, World!” that
many times.

### Examples:
```
Input a number:
5
Hello, World!
Hello, World!
Hello, World!
Hello, World!
Hello, World!
```
```
Input a number:
3
Hello, World!
Hello, World!
Hello, World!
```
```
Input a number:
1
Hello, World!
```
